# RFC: Reputation and Trust Layer for Agora (Archived)

This document is archived to avoid drift with the implementation.

## Canonical sources

- Implementation-oriented RFC and status: `docs/rfc-001-reputation.md`
- High-level architecture and current CLI surface: `README.md`

## Why archived

The previous version of this document contained exploratory designs that diverged from the current shipped behavior and command surface. Keeping it active alongside `rfc-001` caused contradictions in:

- command availability
- scoring and storage assumptions
- implementation status of advanced reputation features

If you want to propose new reputation features, open a new RFC update against `docs/rfc-001-reputation.md`.
